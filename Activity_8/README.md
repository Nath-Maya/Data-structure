Activity_8 - Despacho de camiones (cola FIFO)

Descripción
-----------
Programa en Java que modela el despacho de camiones siguiendo el orden
FIFO (primero en entrar, primero en salir). Cada camión tiene código, placa y
nombre del chofer.

Estructura
----------
- Truck.java      -> clase que representa un camión (registro)
- TruckQueue.java -> clase que maneja la cola FIFO y validaciones (evita duplicados)
- MainApp.java    -> programa principal con menú interactivo en consola
