# Activity_Stack - Registro de artículos (LIFO)

Programa en Java que implementa una pila (stack) para almacenar
artículos no perecederos. Los artículos se ingresan por consola y se muestran
en orden LIFO (último en entrar, primero en salir).

Estructura de archivos:

- Article.java        - Clase que representa un artículo (código, nombre, detalle, precio)
- Node.java           - Nodo simple para la pila
- Stack.java          - Implementación de la pila con operaciones push, pop y display
- Main.java           - Menú interactivo en consola (español)

 (Data-structure). Está en castellano y con mensajes
amigables para la consola.
